# AI Roblox Asset Maker

Turn a text prompt into game-ready art for Roblox - icons, decals, GUI buttons, thumbnails. Generates PNGs you upload straight into Studio.

## Quick start

```bash
pip install -r requirements.txt
cp .env.example .env   # then add your OPENAI_API_KEY
python generate_assets.py "a glowing blue sword icon, flat style"
python generate_assets.py "cartoon coin" --count 4 --size 512x512
```

PNGs land in `./assets`. In Studio: **Asset Manager -> Images** (or insert as a Decal).
